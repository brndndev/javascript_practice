Dog tags

These three characters pertains to the user persons

1st three Owner of file

2nd three User / Group of file

3rd set everyone else

Values

R = 4

W = 2

Permissions expressed in a 3 number format (ex. 755)

444 = Owner, User, and World can read this file

755 = Owner can R/W/X
	   User can R/X
	   World can R/X
